Remalswor

Description of the game

In this game, the main character is "Remalswor". At the beginning of each round, there are a choice of two spells (attacking and heal) after choosing which, a list with 13 different tasks appears. With the correct execution of any chosen task, the main character reproduces the spell, after which, the enemy begins to attack. If the task was a non-correct decision, then the hero's spell is not reproduced, and the monster deals damage. The game continues until the main character is killed. After killing the hero, a table with high scores appears, as well as a "Try again" button.
